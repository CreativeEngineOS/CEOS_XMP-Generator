
# XMP Generator Tool

This is a browser-based metadata generator for creating .XMP sidecar files compatible with Photo Mechanic and similar software.

## How to Use

1. Open `index.html` in any browser or deploy via GitHub Pages.
2. Fill out the form and click "Generate .XMP File".
3. A valid `.xmp` file will be downloaded with your input metadata.

## GitHub Pages Setup

- Go to your repository settings.
- Under 'Pages', choose source: `main` branch / root.
- Visit `https://yourusername.github.io/xmp-generator/` to use it online.
